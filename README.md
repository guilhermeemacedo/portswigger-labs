# PortSwigger Web Security Academy Labs

Este repositório contém writeups e anotações pessoais
baseados nos laboratórios da PortSwigger Web Security Academy.

O objetivo é consolidar conhecimentos práticos em
Application Security, segurança de APIs e vulnerabilidades
comuns encontradas em aplicações web modernas.

## 🎯 Foco de estudo
- Application Security (AppSec)
- API Security
- Authentication & Authorization
- Access Control
- Vulnerabilidades do OWASP Top 10

## 🧪 Fonte
Todos os laboratórios utilizados neste repositório são provenientes da
[PortSwigger Web Security Academy](https://portswigger.net/web-security),
plataforma educacional mantida pelos criadores do Burp Suite.

## ⚠️ Aviso
Este repositório possui finalidade **educacional**.
As vulnerabilidades descritas são exploradas apenas em
ambientes de laboratório controlados.
